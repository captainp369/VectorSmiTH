# Example scenes

Copy any of these over the project's `scene.json` (or paste into it) while
`npm run dev` is running and the canvas will render it instantly:

```bash
cp examples/thai-youtube-thumbnail.json scene.json
```

| file | canvas | shows off |
|------|--------|-----------|
| `thai-youtube-thumbnail.json` | 1280×720 | gradient background/overlay, Thai typography (Kanit), rotated badge |
| `ig-post-starter.json` | 1080×1080 | editorial layout, star accent, line + serif type |

These are also handy as few-shot references when prompting the AI.
